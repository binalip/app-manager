APP Manager Concurrency Test Plan 
---------------------------------
---------------------------------


This test plan is written for plan-your-trip webapp. Test case matches the loginuser name with the jwt user name in the web page. 

There are 2 Thread Groups each per one user.

User credentials are defined in 'User Defined Variables' in each thread group. 

Pre-Requisites
---------------

Deploy samples following the instructions given for them.
Create users with required claims for each app.
